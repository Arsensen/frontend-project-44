## Run the project

### 1. Install dependencies

```bash
npm install
```

### 2. Run a game

Each game is a standalone script under `bin/`. For example:

```bash
node bin/brain-even.js
node bin/brain-calc.js
node bin/brain-gcd.js
node bin/brain-progression.js
node bin/brain-prime.js
```
